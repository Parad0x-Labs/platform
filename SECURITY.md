# Security Policy

## Reporting a Vulnerability

Parad0x Labs takes security seriously. If you discover a potential security vulnerability, please do not use the public issue tracker. Instead, send a detailed report to **hello@parad0xlabs.com**.

We appreciate your help in keeping our ecosystem safe and follow a responsible disclosure policy.

## Supported Versions

Only the latest production releases of the Liquefy Blackbox and Nebula Media engines are supported for security updates.
